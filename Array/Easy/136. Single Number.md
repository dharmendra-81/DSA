##### Given a **non-empty**Â array of integers `nums`, every element appears _twice_ except for one. Find that single one.

```cpp
class Solution {
public:
    int singleNumber(vector<int>& nums) {
        int ans=0;
        for(int i=0;i<nums.size();i++){
            ans ^= nums[i];
        }
        return ans;
    }
};
```
