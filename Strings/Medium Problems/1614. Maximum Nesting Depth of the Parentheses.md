Given a **valid parentheses string** `s`, return the **nesting depth** of `s`. The nesting depth is the **maximum** number of nested parentheses.

---
```cpp
int maxDepth(string s) {
        int p=0;
        int ans=0;
        for(char x : s){
            if(x == '(') p++;
            else if (x == ')') p--;
            ans = max(ans, p);
        }
        return ans;
    }
```
O(N)
